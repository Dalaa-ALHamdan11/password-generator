# 🔐 Random Password Generator

A simple web application that generates secure random passwords using JavaScript.

## 🚀 Features
- Generate random passwords
- Customize password length
- Copy password to clipboard
- Simple and user-friendly interface

## 🛠 Technologies
- JavaScript
- HTML
- CSS

## 📚 What I learned
- DOM manipulation
- Event handling
- Using Math functions
- Clipboard API
